# Assistant-UI Demo

This repository is a practice project using `assistant-ui`.

## Overview

A demo application that implements data analytics features using `assistant-ui`.

## Features

This demo includes several tools with custom UI components for displaying results:

- **display_graph**: Displays various types of graphs (bar, line, pie charts)
- **display_report**: Displays analysis reports with multiple charts
- **search_products**: Displays product search results in a table format
- **get_weather**: Displays weather information in a user-friendly format

## Project Structure

- **agent/**: LangGraph-based backend server
  - See [`agent/README.md`](./agent/README.md) for details

- **frontend/**: Next.js-based frontend application
  - Based on [assistant-ui's with-assistant-transport example](https://github.com/assistant-ui/assistant-ui/tree/main/examples/with-assistant-transport)
  - See [`frontend/README.md`](./frontend/README.md) for details
